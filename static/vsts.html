On many web services teams, performance is a key concern. Latency increases in the tens of milliseconds can over 10% increases in the number of dropped user sessions.

On the MSN Core Rendering team, I used Visual Studio Test System to create load and performance testing suites, allowing data driven unit tests to be used to quickly create suites and generate reports to allow the team to be aware of performance regressions.
